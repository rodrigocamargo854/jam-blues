# Treino Matinal — Piano (jam-blues)

Aplicativo simples para prática matinal de piano, com exercícios de aquecimento, escalas, grooves e progressões (blues, jazz, soul, gospel).

## Stack
- HTML + CSS (estilos inline / estáticos)
- JavaScript (vanilla) para UI e lógica do exercício
- Sem banco de dados (conteúdo estático)

## Regras
- Commits em português.
- Código JavaScript simples e legível; evitar frameworks pesados.
- Priorizar acessibilidade (HTML semântico e labels) e performance.

## Estrutura
- `index.html` → UI principal (exercícios, timers, controles).
- `ai/` → documentação, specs e material para agentes/automação.
- `ai/harness/` → arquivos de contexto para agentes (constitution, learnings, agents).
